Condense is a cloud orchestration tool that enables you to quickly model, deploy, manage and monitor your whole infrastructure

Build

Create/Resize/Delete your servers with a few clicks. Works great with AWS, DigitalOcean, Google Compute Cloud and Microsoft Azure

Provision

Automatically apply your CAPS(Chef, Ansible, Puppet and Salt) provisioning scripts. Condense will run them in parallel and report execution stats in real time.

Manage

Configure services and sync the changes in real time.

Monitor

Have a complete visibility over your infrastructure out of the box - from general system metrics like CPU, Memory, Disk usage and network throughput to service checks like open ports and individual process metrics.
