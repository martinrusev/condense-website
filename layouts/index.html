Build

Model your cloud services with a few clicks. An intuitive, universal interface that works across Amazon Web Services, DigitalOcean, Google Compute Cloud and Microsoft Azure.

Provision

Automatically apply your CAPS(Chef, Ansible, Puppet and Salt) provisioning scripts. Condense will run them in parallel and report execution stats in real time.

Monitor

Have a complete visibility over your infrastructure out of the box - from general system metrics like CPU, Memory, Disk usage and network throughput to service checks like open ports and individual process metrics. Condense automatically syncs officially provided metrics for your services, like Amazon CloudWatch for example.

The monitoring in Condense is built around the solid base already established in Amon.
